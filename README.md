# Farm-Fresh-project
FarmFresh – Organic Grocery Pre-Order & Delivery Website is a multi-page web application developed using HTML, CSS, and JavaScript. The website allows users to explore a variety of organic products, add selected items to a shopping cart, and place pre-orders efficiently.

The project uses structured HTML for page layout, CSS for styling and responsive design, and JavaScript for handling functionality such as adding items to the cart, storing data in localStorage, and managing order processing. All pages, including Home, Products, Cart, and Checkout, are connected through hyperlinks to provide smooth navigation.

The cart system dynamically updates selected items and calculates the total cost using JavaScript. Data is stored in localStorage to ensure that cart items are not lost on page refresh. The checkout page collects user details, delivery preferences, and payment method before confirming the order.

Overall, the project demonstrates a complete e-commerce workflow using core web technologies without relying on frameworks, making it simple, efficient, and easy to understand.
